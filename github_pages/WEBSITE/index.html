项目简介

运用平均场博弈理论和 主体建模方法， 通过大规模数值仿真和参数校准，深入分析就业市场的均衡机制， 为不同政策干预方案提供科学的量化评估。

本系统采用 Numba JIT 加速技术，支持百万级主体仿真， 结合 Plotly 交互式可视化，实现复杂模型的直观呈现。

核心模块

Population 人口初始化

基于真实数据生成异质性个体分布，初始化劳动者和企业的状态变量（T、S、D、W）， 支持多种分布类型和参数设定，为后续模型提供真实的初始人口。

异质性建模 分布拟合 统计检验 数据可视化

Logistic 匹配函数估计

采用逻辑回归估计劳动力市场的匹配函数和分离率函数， 量化个体特征（技能、数字素养等）对就业匹配概率的影响。

匹配函数 分离率 系数估计 拟合优度

MFG 均衡求解

基于 Bellman 方程和蒙特卡洛方法，求解大规模异质性个体的动态博弈均衡， 实时监控收敛过程，并通过3D交互式图表展示价值函数和最优策略。

3D价值函数 策略热力图 人口演化 收敛监控

Calibration 参数校准

采用模拟矩匹配方法，通过差分进化算法实现模型参数的全局优化， 提供参数空间探索、目标函数轨迹和矩拟合质量诊断等功能。

SMM方法 全局优化 质量诊断 敏感性分析

Simulation 政策仿真

模拟不同政策组合下的市场演化路径，量化评估政策效果与成本收益， 支持多场景对比、时间序列预测和异质性效应分解。

多场景对比 时序预测 效应分解 成本收益
