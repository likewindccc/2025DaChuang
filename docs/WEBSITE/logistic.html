Logistic 模块

匹配函数估计

采用逻辑回归估计劳动力市场的匹配函数和分离率函数， 量化个体特征（技能、数字素养等）对就业匹配概率的影响， 为MFG模型提供关键参数。

模块说明

匹配函数（λ）： 刻画劳动者找到工作的概率，取决于个体特征（T、S、D、W）、 企业需求（σ）和劳动力市场紧张度（θ）。

分离率函数（μ）： 刻画就业者失去工作的概率，主要由个体特征与岗位匹配度决定。

技术方法： 基于历史就业数据，使用Logistic回归估计两个函数的系数， 通过ROC曲线、混淆矩阵等指标评估模型性能。

模型公式

匹配函数：
λ(x', σ, θ) = 1 / (1 + exp(-(β₀ + β₁·T + β₂·S + β₃·D + β₄·W + β₅·θ)))
分离率函数：
μ(x, σ) = 1 / (1 + exp(-(γ₀ + γ₁·S + γ₂·D + γ₃·|W-σ|)))
