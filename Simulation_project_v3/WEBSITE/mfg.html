MFG 模块

平均场博弈

基于平均场博弈理论，构建大规模异质性个体动态博弈均衡模型， 刻画劳动者的最优努力策略和就业市场的长期稳态分布。 采用Bellman方程求解个体最优策略，通过Monte Carlo仿真演化人口分布。

理论基础

平均场博弈（MFG）： 一种分析大规模主体交互系统的理论框架，由Lasry-Lions (2006)和Huang-Caines-Malhamé (2006)独立提出。 核心思想是用平均场分布代替所有其他个体的影响，将多主体博弈简化为单个代表性主体对平均场的优化问题。

求解策略： 采用不动点迭代法求解MFG均衡，交替求解Bellman方程（个体最优策略）和 Kolmogorov Forward方程（人口分布演化），直至收敛。

适用场景： 大规模异质性劳动力市场、交通流量、能源市场、金融市场等需要刻画大量主体交互的复杂系统。

核心方程

Bellman方程（个体最优策略）：

VU(x) = maxa { b0 - κ·a² - αT·(T-Tideal)² + ρ·[λ(x')·VE(x') + (1-λ(x'))·VU(x')] }
状态转移方程：

x' = x + γ·a·(xbound - x)
人口分布演化（Monte Carlo仿真）：

根据最优策略a*和转移概率，模拟10000个个体的状态演化

求解算法

  1. 初始化人口分布
    从Population模块获取10000个异质性个体，初始化失业率u0
  2. 求解Bellman方程
    使用值迭代算法计算失业价值VU、就业价值VE和最优努力策略a*
  3. 演化人口分布
    根据a*和转移概率λ、μ，Monte Carlo仿真个体状态演化，更新人口分布
  4. 计算新失业率
    统计新分布下的失业人数，计算ut+1
  5. 检查收敛
    判断|ΔV|/|V| < εV、|Δmean(a)| < εa、|Δu| < εu
  6. 输出均衡结果
    保存最优策略、价值函数、人口分布、失业率等关键指标
